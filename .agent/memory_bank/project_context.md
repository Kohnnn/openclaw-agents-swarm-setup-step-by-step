# Project Context

## High-Level Goals
- Create a fully no-code friendly documentation repository on GitHub for setting up an Agent Swarm using OpenClaw.
- The swarm will automate daily workflows and support multi-platform chat integration.

## User Stories & Use Case
- **Initial Use Case**:
  - Orchestrator Agent: Brainstorms what to do.
  - Sub-agents (1, 2, 3): Perform the tasks.
  - Reviewer Agent: Receives the results from sub-agents, reviews the work, and then commits/pushes the code to the user.
- **Multi-Platform Integrations**:
  - Agents will be controlled via popular chat interfaces (Discord, Telegram, Slack, WhatsApp).
  - Multiple chat rooms/spaces for AI-to-AI and AI-to-User communication.
  - Users can communicate and upload data to the bots via the platform of their choice.
- **Onboarding/Documentation**:
  - A comprehensive, step-by-step `README.md`.
  - Content includes: How to install OpenClaw/ZeroClaw, how to connect with messaging apps, and how to run multiple instances/agents on a single computer/run.
  - The documentation will creatively combine the Swarm execution flows with varied real-world use cases (e.g., social media, coding, productivity).
  - Use our custom hosted architecture and logic images throughout the documentation.

## Business Requirements
- The documentation must be beginner-friendly ("no code friendly").
- Provide clear examples and guidelines.
